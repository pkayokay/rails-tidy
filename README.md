# Rails Scripts

A collection of handy scripts for Rails apps. Copy these into the same paths in your Rails project (e.g. `bin/` scripts go in `bin/`, `lib/tasks/` files go in `lib/tasks/`).

## Scripts

| Script | Description |
|---|---|
| [Hatchbox](docs/hatchbox.md) | SSH into Hatchbox-deployed Rails apps, open consoles, and tail logs |
| [Telegram Deploy Notification](docs/telegram.md) | Send a Telegram message with commit info after each deploy |
| [Tidy](docs/tidy.md) | Run Prettier, Herb formatter, and Herb linter in parallel |
